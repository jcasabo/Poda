Poda de renovación

Se trata de una poda drástica, que solo se realiza con fines muy específicos:

  • Para preparar el árbol antes de injertarlo para cambiar la variedad de cultivo.
  • En las especies que rebrotan con vigor, para rejuvenecer el árbol. En este caso, puede eliminarse toda la copa, dejando solo las ramas principales; o ir renovando escalonadamente, de forma que cada año eliminamos la copa de una de las ramas principales. Este último método resulta menos agresivo para la planta.
  • En caso de alguna enfermedad que solo puede ser tratada eliminando las partes afectadas.
